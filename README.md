# timeline-task

## Деплой

Приложение раскатано [здесь](https://alexandr-zabelin.github.io/timeline-task/)

## Локальный запуск

Запустить локально можно:

- При наличии докера - через него:

`docker-compose up`

- Вручную:
  `npm i`
  `npm start`

В обоих случаях проект будет доступен на http://localhost:8080/
